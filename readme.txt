This is a badge generator plugin for Buddypress.

== Description ==

This is a badge plugin that generate user's badge for link back to buddypress user profile page from other website or social network.
It includes functions that make badge image from website logo and option to attach xprofile information and profile image.

change badge logo by replace logo.png in /images directory, badge width will change to width of logo image
 
== Installation ==

1. Upload buddypress-badge to the /wp-content/plugins/ directory or use automatic installation from wp plugin panel
2. Activate the plugin through the 'Plugins' menu in WordPress
